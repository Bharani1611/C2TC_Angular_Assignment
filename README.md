## 🚀 Running the Application


### Steps to Run

1. **Clone/Navigate to Project**:

   ```bash
   cd student-dashboard
   ```

2. **Install Dependencies**:

   ```bash
   npm install
   ```

3. **Start Development Server**:

   ```bash
   ng serve
   ```

4. **Open Browser**:
   Navigate to `http://localhost:4200`

5. **Build for Production** (Optional):
   ```bash
   ng build
   ```
